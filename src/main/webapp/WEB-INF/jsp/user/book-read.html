“亲爱的薇薇，CK国际大酒店1026号房，晚上8点，去后关上灯，等我，我会给你一个惊喜哦！”

慕薇薇默念着手机短信上男朋友陆子航发来的信息，笑出了声来，此刻的她已经在这个房间了。

放下东西，配合的关上了房间里的灯，一片黑暗笼罩了下来，慕薇薇摸索着坐在了旁边的沙发上静等，也不知道陆子航会给她什么样的惊喜呢？该不会是要求婚吧？

两个人交往已经有一年半了，对于这个男人，她心底里是喜欢的，只是总觉得缺少点什么。

慕薇薇正想着，只听“砰——”地一声，好像是有什么东西撞到门板的声音，紧接着就是男人轻微的闷哼声。

吓——

慕薇薇顿时整个人都呆住了，没看见有人刷卡进了房间啊，那这个声音是——

想到这里，她立即背脊发凉，站起了身来，就想要伸手去摸墙壁上的灯开关。

然而，慕薇薇的手刚触碰到墙壁，就被旁边一只温热的手掌抓住了，男人低沉沙哑的声音随即响了起来，“呵，女人……”

天呐！！！

真的有人！！！

他是怎么进来的？难道刚刚就一直躲在房间里吗？

“啊……”还没等慕薇薇来得及多想什么，男人就一把抓住了她的手腕，将她拉进了自己的怀里。

男人另外一只手直接在慕薇薇的背后一扯，她身上穿着的连衣裙就被他扯了下来。

“天呐，救命……”慕薇薇下意识的呼唤，浑身发抖，用手想要护住胸前。

这个男人绝对不是陆子航，他到底是谁？

男人速度极快，直接拦腰一抱，就将慕薇薇整个人抱了起来。

“你……你是谁？流氓，快放开我！”慕薇薇颤栗不已，她浑身紧绷着，身体贴着男人。

他的身上很热，不是一般的热，她贴着他的皮肤仿佛快要被他烧着了一般。

“我是谁？”男人念了念这三个字，似乎轻笑了下，声音沙哑又隐忍的说道，“马上你就知道我是谁了！”

他的话音刚落，借着窗外依稀透进来的亮光，看到了床的位置，抱着慕薇薇便往床边走去，随即一把将她丢到了大床上……

慕薇薇被吓得发抖，立即就想逃走，男人却根本不给她这个机会，直接一把扑了过来，将她整个人压在了身下。

“唔……”

他火热的唇压了下来，狠狠的啄着她的唇瓣，唇齿纠缠，慕薇薇挣扎，奈何男人的力气大得惊人，她竟然推都推不动他。

“混……蛋，唔……放开……”

“呵，一会儿你就不想让我放开了……”男人禁锢住慕薇薇，压了下来，他不再隐忍，轻声道，“配合我，帮我降火……”

“谁要帮你，混蛋！”慕薇薇骂道，十分屈辱，眼泪落了下来，泪如雨下，却被男人悉数吻去了。

他的声音温柔又沙哑，“我会很轻的。”

一个小时后……

男人终于释放了，慕薇薇反抗不过，已经哭得没有眼泪了，当他放开她的时候，她立即从床上爬了起来。

“你这个混蛋，我不会放过你的！”慕薇薇说着，伸手正好抓到了自己的连衣裙和包包，她立即就往身上套。

借着房间里并不清晰的光亮，慕薇薇勉强看清了眼前的男人，一米八零的个子，挺拔有力的身材，五官的轮廓线条很完美，一双眼睛幽暗而深邃，但灯光实在太暗，她没能看清楚他的脸，只依稀感觉应该是个异常英俊帅气的男人。

“整个A市，想要弄死我的人多得去了，不过，我期待你的反击！”男人勾唇轻笑道，似乎是想到了方才的美妙滋味，某处有了感觉，他舔了舔唇瓣，“在此之前，我们可以再来一次！”

说着，就想要将慕薇薇抓回来。

慕薇薇吓得立即跳下了床去，嘴里大叫道，“混蛋，你去死吧！！！”

说着，她不敢再耽搁，拿起包包，立即就借着光亮，摸索着往房间门口走去。

“不要走，告诉我你是谁？”察觉到慕薇薇的目的，男人大喊道，立即起身便想要去追。

“我是你小姑奶奶！”

慕薇薇冷笑道，拉开房间门就逃了出去，同时还故意将门打得大开，因为她知道他并没有穿衣服，此刻光溜溜的，不可能会追出来。

不敢乘电梯，慕薇薇从楼梯间跑了下去，同时不停的注意着周围，好在那个男人真的没有追出来。

只是，那个男人，夺走了她第一次的男人……到底是谁？
